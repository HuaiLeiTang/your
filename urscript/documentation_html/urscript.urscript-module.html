Module urscript

source code

script.py module contains classes and functions for generating UR Script programs

It contains:
    1) Functions for generating, loading and saving UR Script programs/functions
    2) Functions for generating UR Script statements (commands,comments, expressions)
    3) Functions for generating Pose type that is specific to UR script.

action(pose, id, on, sleep_time=0.5, retract=0.0, accel=1.2, vel=0.3)

source code 

Returns a tuple of UR script commands for a motion-digital out compound action Robot moves to target pose and sets digital out. An optional retraction motion is added at the end. For example, this is used for pick/place operations. Args: pose: Target pose (can also be specified as joint positions) id: The number (id) of the digital output. (int) on: The signal level. (boolean) sleep_time: Optional. Wait time following digital_out [s] retract: Optional. Distance to retract in TCP's -z direction after sleep. [m] accel: Optional. Tool accel [m/s^2] vel: Optional. Tool speed [m/s] Returns: Tuple of UR script commands
